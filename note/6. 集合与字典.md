# 第六章 集合与字典

## 一、集合及其表示

### 1. 概念

- 集合是成员（元素）的一个群集。集合中的成员可以是原子（单元素），也可以是集合
- 集合的成员必须**互不相同**
- 同一集合中所有成员具有**相同的数据类型**
- 集合成员一般是**无序的**

### 2. 位向量的实现方法

**存储结构：**

`unsigned short* bitVector`：一个`unsigned short`类型的一个数组，数组中的每个元素是一个16位的二进制数。

`bitVector[i]`的第`j`位可以是`0`或`1`，表示$i \times 16 + j$是否在集合之中。

**访问方式：**

确定在第几个元素里：`ad = x / 16​`

确定在元素的第几位：`id = x % 16`​

返回这个位的数字：`number = bitVector[ad] >> id % 2` 通过右移把这个数字移到个位上，然后对2取模

#### 部分关键函数的实现

**1. 判断元素是否存在**

```cpp
// 取集合元素x，判断元素x是否在集合中，如果是返回1，否则返回0
int getMember(const T x) const {
    if (x < 0 || x > setSize) return 0;
    int ad = x / 16;
    int id = x % 16;
    unsigned short elem = bitVector[ad];
    return ((elem >> id) % 2);
}
```

**2. 给指定元素设值**

```cpp
// 把值v送入集合元素x中
void putMember(const T x, int v) {
    if (x < 0 || x > setSize) return;
    int ad = x / 16;
    int id = x % 16;
    if (v == 1) {
        bitVector[ad] |= (1 << id);
    } else {
        bitVector[ad] &= ~(1 << id);
    }
}
```

**3. 判断是否R的子集**

```cpp
// 判断集合 this 与 R 相等
bool operator==(const bitSet<T> &R) {
    if (vectorSize != R.vectorSize) return false;
    for (int i = 0; i < vectorSize; i++) {
        if (bitVector[i] != R.bitVector[i]) {
            return false;
        }
    }
    return true;
}
```
