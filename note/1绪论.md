## 4. 算法和算法分析

时间复杂性与空间复杂性

算法必须满足的五个重要特性

1. 输入
2. 输出
3. 确定性
4. 有穷性
5. 有效性

时间复杂性度量：
用**程序步**来衡量一个程序的执行时间。
赋值的程序步 = 1
循环程序步 = 循环次数 * 循环内部程序步

算法的（渐近）时间复杂度：$T(n) = O(f(n))$

```cpp
for(int i = 0; i < n; i++){         // n+1
    for(int j = 0; j < n; j++){     // n(n+1)
        c[i][j] = 0;                  // n^2
        for(int k = 0; k < n; k++){     // n^2(n+1)
            c[i][j] += a[i][k] * b[k][j];   // n^2(n+1)
        }
    }
}
```
